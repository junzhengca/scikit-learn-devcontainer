# scikit-learn devcontainer

A simple Docker development environment for scikit-learn.

You must have

* Docker
* Visual Studio Code
* [Remote Development Extension Pack](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.vscode-remote-extensionpack)

## Get Started

1. Clone this repo
2. `./setup.sh <your fork>`, ie `./setup.sh yifeiyin`
3. Open VSCode with the folder `./scikit` if it's not open already â€“ the script will open vscode for you if the command `code` is available
4. Run `./.devcontainer/setup.sh` **inside the container**
